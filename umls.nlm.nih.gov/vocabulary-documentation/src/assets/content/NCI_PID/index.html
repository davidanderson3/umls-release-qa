Authority
The Nature Pathway Interaction Database is produced and maintained by the Nature Publishing Group (NPG) and the U.S. National Cancer Institute (NCI).

Purpose
The Nature Pathway Interaction Database serves as a reference source of biomolecular interactions and for use to run computational analyses.

Description
The database is an NCI Thesaurus (NCIt) property that consists of identification codes for pathways; links to information about known biomolecular interactions and key cellular processes assembled into signaling pathways. It can be used by the NCIt Browser to link a pathway concept to the source pathway information.

Audience
The Nature Pathway Interaction Database is employed by the cancer research community and others interested in cellular pathways, such as neuroscientists, developmental biologists, and immunologists.

Metathesaurus Update Frequency
The Nature Pathway Interaction Database is updated annually as part of the NCI Thesaurus in the Metathesaurus.
