Description
The Content Archive Resource Exchange Lexicon (CareLex) is an open source content classification model for the BioPharmaceutical industry. The model supports content classification for the industry and facilitates interoperability through a standards-based approach.

Metathesaurus Update Frequency
CareLex is updated annually as part of the NCI Thesaurus in the Metathesaurus.
