VSAB: NEU2013_06_09

Summary of Changes

1. Non-English synonyms (except for Latin) are now mapped to the new ATN = TRANSLATION. The ATV has the format "language: string" (e.g., TRANSLATION|Italian: encefalo).

2. Latin synonyms are considered scientific names so they are treated as English strings and assigned the TTY=SY.

3. New TTY = OP (obsolete preferred term) and new TTY = IS (obsolete synonym). If the preferred term is obsolete it is assumed that all related synonyms are also obsolete.

4. New ATN: TRANSLATION - name of the neuroanatomical structure in various languages.

5. Deprecated TTYs: CHISY, FRESY, GERSY, INDSY, ITASY, KORSY, LATSY, RUSSY, SPASY, TURSY.
