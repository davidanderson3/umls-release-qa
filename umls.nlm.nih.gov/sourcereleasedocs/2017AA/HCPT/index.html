Authority
The HCPCS Version of Current Procedural Terminology® (HCPT) is produced by the Centers for Medicare and Medicaid Services (CMS).

Description
CPT codes are compiled in Level 1 of the Healthcare Common Procedure Coding System (HCPCS). For more information about HCPCS, see the HCPCS source synopsis. For more information about CPT, see the CPT source synopsis.

Update Frequency
HCPT is updated annually.

Metathesaurus Update Frequency
HCPT is updated annually in the Metathesaurus, in conjunction with HCPCS.
