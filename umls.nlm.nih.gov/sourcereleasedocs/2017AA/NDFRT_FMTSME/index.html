• Synopsis

Authority
The National Drug File - Reference Terminology (NDFRT) - Federal Medication Terminologies Subject Matter Expert (FMTSME) relationships were developed by the Federal Medication (FedMed) interagency collaboration. FedMed is organizing an agreed set of standard, comprehensive, freely and easily accessible FMT to improve the exchange and public availability of medication information. The Veterans Health Administration (VHA) maintains the NDFRT-FMTSME relationships within NDFRT.

Purpose
NDFRT-FMTSME relationships are used to associated NDFRT ingredients with NDFRT drug classes.

Description
NDFRT-FMTSME is a set of relationships between NDFRT concepts. No source atoms are included.

Update Frequency
NDFRT-FMTSME is updated monthly.

Metathesaurus Update Frequency
NDFRT-FMTSME is updated in each Metathesaurus release as part of RxNorm..
