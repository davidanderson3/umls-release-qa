VSAB: SCTUSX_2013_03_01

Summary of Changes

Some file names for the SCTUSX release have changed, however there are no changes to the format or processing.

Original Files:

The RF1 format files were processed for SCTUSX_2013_03_01.  Metathesaurus SCTUSX data is extracted from the following tab-delimited files:

SnomedCT_Release_US1000124_20130301/RF1Release/Terminology/Content:
  • sct1_Concepts_ExtensionNational_US1000124_20130301.txt
    sct1_Descriptions_Extension-en_US1000124_20130301.txt
    sct1_Relationships_ExtensionNational_US1000124_20130301.txt
SnomedCT_Release_US1000124_20130301/RF1Release/Subsets/Language-en-US/
  • der1_SubsetMembers_Extension-en-US_US1000124_20130301.txt
    der1_Subsets_Extension-en-US_US1000124_20130301.txt
Note:  The SCTUSX hierarchy is designed as an extension to the SNOMEDCT hierarchy.  Due to complications in representing a mixed-source hierarchy, the SCTUSX hierarchy is not represented in MRHIER.RRF, however it is represented in MRREL.RRF as follows: 
  • REL/REL_INVERSE:  PAR/CHD
  • RELA/RELA_INVERSE:  inverse_isa/isa

Identifiers

Identifiers are assigned as follows:
  • CODE: CONCEPTID
  • SAUI: DESCRIPTIONID
  • SCUI: Same as CODE
  • SDUI: Not Applicable
