Summary of Changes

1) New relationship | relationship attribute(RELA) | RELA inverse:

  • RO|has_indirect_morphology|indirect_morphology_of

Source Provided Files:

The Metathesaurus uses the following four pipe-delimited MEDCIN files:
  • UMLS_MEDCIN_CONCEPTS2012-12-14.TXT
  • UMLS_MEDCIN_DESCRIPTIONS2012-12-14.TXT
  • UMLS_MEDCIN_HIERARCHY2012-12-14.TXT
  • UMLS_MEDCIN_SNOMED2012-12-14.TXT

Identifiers:

Identifiers are assigned as follows:
  • CODE: MEDCIN ID
  • SAUI: Not Applicable
  • SCUI: MEDCIN ID
  • SDUI: Not Applicable
