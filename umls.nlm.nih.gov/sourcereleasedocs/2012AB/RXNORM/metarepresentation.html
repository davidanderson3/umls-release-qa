VSAB:  RXNORM_12AA_120904F

Notes:

RxNorm, a standardized nomenclature for clinical drugs, is produced by the National Library of Medicine. Data files are released monthly and available by download. Within each Metathesaurus cycle, RxNorm is updated multiple times. This document reflects the latest cumulative update. The scope of RxNorm is determined by the combined scope of its source vocabularies which include the following:

Note: Selected portions of MSH and SNOMEDCT are included in the RxNorm release files, however, they are not processed as part of RxNorm for the Metathesaurus except in assigning RXCUI and RXAUI attributes. All MSH or SNOMEDCT data in the Metathesaurus comes directly from that source's data files. This document describes how the RxNorm release files (RXN*RRF) are processed for representation in the UMLS Metathesaurus.  It does not cover processing of individual source vocabularies for inclusion in RxNorm. 

Summary of Changes

No changes were made to the RXNORM format itself or to Metathesaurus source processing for 2012AB.  Data changes include the following:

1. New ATN = RXTERM_FORM. The RxTerm dose form name for this drug. SAB = RXNORM.

2. New RELA|RELA inverse:
  • contraindicated_with_disease|disease_with_contraindication
  • has_contraindicated_drug|drug_contraindicated_for
     Note: RELA = has_contraindicated_drug replaces the dropped RELA = contraindicated_drug.

3. SAB = NDDF, formerly known as NDDF Plus, is now called FDB MedKnowledge.

4. Homeopathic atoms from SAB = MTHSPL are included in this release.

5. MTHFDA is included in each Metathesaurus release as part of RxNorm, however, it was not updated in this version. MTHFDA was last updated in 2011.

Atoms (MRCONSO):

Term Type

Term types are extracted from field 13 of RXNCONSO.RRF and RXNCONSOOCD.RRF.

Suppressibility

For many RxNorm and subsource termtypes, the suppressibility value can be either "O" or "N". For all others, the value is "N". Suppressibility values are extracted from field 17 of RXNCONSO.RRF and RXNCONSOOCD.RRF.

In addition, source NDC attributes may have SUPPRESS="O". 


Attributes (MRSAT):

  (return to top)

All attribute names and their values are extracted from field 9 and 11 of RXNSAT.RRF and RXNSATOCD.RRF except for RXCUI and RXAUI which are created using RXNCONSO.RRF.
