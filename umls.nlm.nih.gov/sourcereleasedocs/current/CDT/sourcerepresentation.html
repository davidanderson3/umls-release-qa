Summary of Changes:

New Class value = 99 in the source data indicates that the entire CDT Code entry is no longer valid (i.e., deleted) in this version. Assigned new TTY = OP (obsolete preferred term) for these entries.

Notes:

CDT in the Metathesaurus is extracted from a file provided by the American Dental Association (ADA).  The Metathesaurus also includes the HCPCS version of CDT, which has RSAB = "HCDT. The CDT hierarchy is created from the ascii data file provided by the ADA, while the HCDT hierarchy is created during Metathesaurus production using analysis of the HCPCS printed manual.

Identifies the type of change made to this version of the Code (i.e., changes from the prior version), with the following meanings:
  • 30 = No change to existing code, nomenclature or descriptor
  • 31 = New procedure code and nomenclature, and descriptor, if present
  • 35 = Revision in the nomenclature or descriptor (or both), or to indicate a revision to an entire subcategory
  • 99 = Entire CDT Code entry is no longer valid (i.e., deleted) in this version.
