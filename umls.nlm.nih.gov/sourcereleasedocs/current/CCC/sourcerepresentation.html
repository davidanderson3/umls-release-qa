Notes

CCC (Clinical Care Classification) was previously labeled "HHC" (Home Healthcare Classification) in the Metathesaurus. CCC2_5 is an update to HHC2003.

Summary of Changes:

  (return to top)

Changes from HHC2003 to CCC2_5:

Assigned "structural" termtypes (HT, PT) instead of "semantic" termtypes (CO, DX, IV):
  • Deleted TTYs: CO, DX, IV
  • New TTYs: PT, HT, MTH_HT
Separated diagnoses and interventions in hierarchy (required new TTY, MTH_HT for "placeholder" atoms)

Source-Provided Files: Summary

  (return to top)

The CCC distribution includes the following files. This data, along with additional information can be accessed at http://www.sabacare.com/

CCC_Diagnoses_HL7_V2.5_21SEPT11.xlsx
CCC_Interventions_HL7_V2.5_Version-A_21Sept11.xlsx
CCC_Outcomes_HL7_V2.5_ 21Sept11.xlsx


These files were converted to text.  Metathesaurus source processing included some data and format clean up.

Additional files of components and modifiers were created for inclusion in the Metathesaurus. 

Not included: 

The "outcomes" file was not directly processed.  The relationship between modifiers and diagnoses are used to indicate an actual or expected outcome.

Additional notes:

Two additional "placeholder" concepts are created during Metathesaurus source processing to build the hierarchy:

    TTY = MTH_HT
    MRCONSO.STR:  Interventions
    MRCONSO.STR:  Diagnoses   

Relationships are created between:
       Diagnoses and appropriate modifiers: 
                REL=AQ;  RELA=actual_outcome_of/has_actual_outcome

                REL = AQ;  RELA = expected_outcome_of/has_expected_outcome

      Interventions and appropriate modifiers:
            REL = AQ;  RELA = modifies/modified_by
