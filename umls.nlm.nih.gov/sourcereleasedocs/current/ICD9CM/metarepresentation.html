VSAB: ICD9CM_2012, MTHICD9_2012

Summary of Changes

No changes have been made to the format of original data files or to UMLS source processing.

Notes on the data

Representation decisions may be subject to analysis and interpretation based on content and/or page layout, including;
  •  The fully-specified name of the string:  There is no official string (atom_name) associated with a given code.  For example, the line "143.0  Upper gum" appears in the data, however the PT string "Malignant neoplasm of upper gum" is created based on implied context.
  •  Whether to represent data as an atom (MTHICD9/ET) or attribute ("SOS|Includes")

Source Provided Files:

ICD9CM in the Metathesaurus was originally processed in 1999 using a combination of machine-processing and human review.  The Metathesaurus is updated annually using the addenda, which are released in .pdf format and downloaded from the following sites:
    Diseases:  http://www.cdc.gov/nchs/icd/icd9cm_addenda_guidelines.htm#addenda
    Procedures:  https://www.cms.gov/ICD9ProviderDiagnosticCodes/04_addendum.asp 

In addition, short forms (or abbreviations) are downloaded annually from https://www.cms.gov/ICD9ProviderDiagnosticCodes/06_codes.asp#TopOfPage and extracted annually from the following files:
  CMS<VER>_DESC_SHORT_DX.txt
  CMS<VER>_DESC_SHORT_SG.txt

For 2011AB (ICD9CM_2012), the following files were used:
    http://www.cdc.gov/nchs/data/icd9/ICD-9-CM%20TABULARADDENDAfy12.pdf
    https://www.cms.gov/ICD9ProviderDiagnosticCodes/downloads/FY2012_Addenda.pdf   
    CMS29_DESC_SHORT_DX.txt
    CMS29_DESC_SHORT_SG.txt

Identifiers:

Identifiers are assigned as follows:
  • CODE: ICD9CM code value
  • SAUI:  Not applicable
  • SCUI:  Not applicable
  • SDUI: same as CODE
Note:  MTHICD9 "ET" atoms are closely related but not necessarily synonymous with their ICD9CM "PT" or "HT."  Therefore, they are assigned the same SDUI as the ICD9CM "PT" or "HT" with the same code.
