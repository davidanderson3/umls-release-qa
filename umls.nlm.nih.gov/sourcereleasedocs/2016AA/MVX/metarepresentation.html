VSAB: MVX2015_07_27_16_02_24

Summary of Changes

No changes were made to the source data format. The content for this version remains the same, however it was re-proccessed. Based on a clarification from the CDC, processing of manufacturers with a status of "Inactive" was updated. All MVX codes are considered "active."

This results in the following change to processing when the value of the status field is "Inactive":
Previously: TTY = OP; Suppress = "O"
New: TTY = PT; Suppress = "N"
