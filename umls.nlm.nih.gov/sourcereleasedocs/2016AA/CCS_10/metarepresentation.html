VSAB: CCS_10_2016

Summary of Changes

CCS_10 is a new source. It is based on the CCS for ICD-9-CM.

Source Provided Files:

The input files are in standard comma-separated-values format, with fields descriptions in the first line.

File Name Description
ccs_dx_icd10cm_2016.csv Diagnosis categories and mappings
ccs_pr_icd10pcs_2016.csv Procedure categories and mappings
