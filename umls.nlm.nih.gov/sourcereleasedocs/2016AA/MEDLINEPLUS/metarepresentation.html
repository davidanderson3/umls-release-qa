Source Provided Files:

The UMLS uses MedlinePlus xml files downloaded from http://www.nlm.nih.gov/medlineplus/xmldescription.html

For 2016AA, the files used were:
  • mplus_topics_2015-10-21.xml
  • mplus_topic_groups_2015-10-21.xml
