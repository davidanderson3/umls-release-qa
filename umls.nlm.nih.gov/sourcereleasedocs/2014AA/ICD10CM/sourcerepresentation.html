Notes

  • The Metathesaurus uses files submitted in TREF (Terminology Representation and Exchange Format). These files are produced by the National Center for Health Statistics (NCHS) for Metathesaurus processing, and are not currently available publically. TREF was developed by the NLM and is a subset of the Rich Release Format (RRF) Â– the format in which the UMLS Metathesaurus is released. There is now an xml file available for ICD-10-PCS, which was processed and compared to the TREF data. Discrepancies in the data were addressed in the TREF files.
  • Additional files and documentation are downloaded from http://www.cms.gov/Medicare/Coding/ICD10/2014-ICD-10-CM-and-GEMs.html.
