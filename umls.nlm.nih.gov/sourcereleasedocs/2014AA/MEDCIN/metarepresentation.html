VSAB: MEDCIN3_2013_12_18

Summary of Changes

Ascii forms of strings generated from non-ascii strings are no longer being created. MTH* atoms along with their associated relationships have been deprecated (e.g., MTH_* atoms, SFO/LFO|mth_plain_text_form_of).

Source Provided Files:

The Metathesaurus uses the following four pipe-delimited MEDCIN files:
  • UMLS_MEDCIN_CONCEPTS2013-12-18.TXT
  • UMLS_MEDCIN_DESCRIPTIONS2013-12-18.TXT
  • UMLS_MEDCIN_HIERARCHY2013-12-18.TXT
  • UMLS_MEDCIN_SNOMED2013-12-18.TXT

Identifiers:

Identifiers are assigned as follows:
  • CODE: MEDCIN ID
  • SAUI: Not Applicable
  • SCUI: MEDCIN ID
  • SDUI: Not Applicable
