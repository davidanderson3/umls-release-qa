Notes

Summary of Changes:

  (return to top)

There have been significant changes made to the structure of the Perioperative Nursing Data Set (PNDS) and the way codes are assigned since the previous was processed for inclusion in the Metathesaurus in 2002.  Previously, this source included outcomes, diagnoses, interventions, and domains, which were assigned term types of PS/PX, DX, IV, and HT, respectively.  In the current verison, there are assessments, evaluations, implementations, outcomes and domains, and term type assignments have changed accordingly.  Domains are still assigned TTY = "HT," as are placeholder atoms for "Assessment," "Evaluation," "Implementation," and 'Outcome," which were created for the purpose of representing a navigational hierarchy.  All other PNDS atoms are assigned TTY = "PT." 

The domains are the same and have the same codes as the previous version, but the code assignment for everything else has changed, and the strings have often changed also.

Documentation and Reference

The electronic distribution of PNDS did not include documentation or reference material.  However, the printed book, "Perioperative Nusing Data Set 3rd Edition" may provide additional insight into the content and usage of PNDS and information is available here:  http://www.aorn.org/PracticeResources/PNDSAndStandardizedPerioperativeRecord/
