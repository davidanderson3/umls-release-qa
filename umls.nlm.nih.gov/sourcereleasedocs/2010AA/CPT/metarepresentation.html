VSAB: CPT2010, MTHCH2010

Summary of Changes

No changes have been made to the format of original data files. During UMLS source processing, certain relationships are generated between CPT concepts based on string parsing. For example, "Anesthesia for closed chest procedures; needle biopsy of pleura" (code 00522) is given an RO relationships to "Anesthesia for closed chest procedures; pneumocentesis"(code 00524). The algorithm to create these relationships has been changed slightly, resulting in a greater number of relationships being generated. These relationships are not submitted or reviewed by the American Medical Association (AMA), and are attributed to SAB="MTHCH".

Original Files:


The UMLS uses tab-delimited CPT files submitted by the AMA. The files used for 2010 are:
  • LONGULT.txt
  • MEDU.txt
  • MODUL.txt
  • SHORTU.txt

In addition, updates were extracted from "CPT 2010 Adjunct File.xls," and "PQRI 2010 Adjunct File.xls."

Identifiers


Identifiers are assigned as follows:
  • CODE: For CPT atoms: The CPT Code in the appropriate file (e.g. in LONGULT.txt for a PT).
    For MTHCH atoms: An MTHU# is generated during Metathesaurus source processing
  • SAUI: Not Applicable
  • SCUI: Same as CODE
  • SDUI: Not Applicable
