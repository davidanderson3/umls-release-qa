VSAB: MEDCIN3_2009_12_30

Summary of Changes

No changes have been made to the format of original data files. In the MEDCIN to SNOMEDCT mapping relations, one new linkage concept, "Grade (attribute)", is used. The UMLS RELA values grade_of and has_grade are used to represent these relations.

Source Provided Files:


The Metathesaurus uses the following four pipe-delimited MEDCIN files:
  • UMLS_MEDCIN_CONCEPTS2009-12-30.TXT
  • UMLS_MEDCIN_DESCRIPTIONS2009-12-30.TXT
  • UMLS_MEDCIN_HIERARCHY2009-12-30.TXT
  • UMLS_MEDCIN_SNOMED2009-12-30.TXT

Identifiers:


Identifiers are assigned as follows:
  • CODE: MEDCINID
  • SAUI: Not Applicable
  • SCUI: Same as CODE
  • SDUI: Not Applicable
