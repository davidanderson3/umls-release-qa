Description
NCI_caDSR (Cancer Data Standards Registry and Repository) defines a comprehensive set of standardized metadata descriptors for cancer research data, for both information collection and analysis. It is designed to integrate Cancer Common Ontologic Representation Environment (caCORE) infrastructure and supports the development and deployment of data elements that are used as metadata descriptors, primarily for NCI-sponsored research.

Metathesaurus Update Frequency
NCI_caDSR is updated annually as part of the NCI Thesaurus in the Metathesaurus.
