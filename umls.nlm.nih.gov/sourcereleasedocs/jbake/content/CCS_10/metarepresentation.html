Summary of Changes

There are no changes to the original data format or to UMLS source processing.

Source Provided Files:

The input files are in standard comma-separated-values format, with fields descriptions in the first line.

File Name Description
CCSCategoryNames(FullLabels).pdf Full labels for single-level and multi-level CCS Categories
ccs_dx_icd10cm_2018_1.csv Diagnosis categories and mappings

Mappings (MRMAP):

Field Value Origin
RELA classified_as Connect ICD-10-CM or ICD-10-PCS with CCS CATEGORY and MULTI CCS LVL 2 atoms. There is overlap between some CCS CATEGORY and CCS LVL 1 CODE values. To disambiguate, note that TOEXPR values do not correspond to MULTI CCS LVL 1 CODES.
