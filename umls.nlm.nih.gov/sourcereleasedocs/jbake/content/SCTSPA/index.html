Description
This is the Spanish translation of the descriptions and documentation from the SNOMED CT® International Release. The current SNOMED CT Spanish is always based on the previous SNOMED CT International Release.

UMLS licensees may download the SNOMED CT Spanish Release files.

For more information about SNOMED CT in the Metathesaurus, see the SNOMED CT source synopsis.

Update Frequency
SNOMED CT Spanish is updated in July and October.

Metathesaurus Update Frequency
