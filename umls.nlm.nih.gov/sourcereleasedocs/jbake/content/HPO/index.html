Purpose
HPO allows for large-scale computational analysis of the human phenome.

Description
HPO describes clinical abnormalities. It includes an is-a hierarchy, as well as broader, narrower, related and exact entry terms and synonyms. Each code is assigned to one of the five subontologies:

  • Phenotypic abnormality,
  • Mode of inheritance,
  • Clinical modifier
  • Clinical course,
  • Frequency
