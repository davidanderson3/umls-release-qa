VSAB: CVX2014_12_29

Summary of Changes:

No changes were made to the source data format. However, processing vaccines with a status of "Inactive" was updated to reflect a clarification from the CDC. All CVX codes are considered 'active' except for when the vaccine status is 'NEVER ACTIVE', or the CVX code is 99, or 999.

This results in the following change to processing when the value of the status field is "Inactive":
Previously: TTY = OP/OA; Suppress = "O"
New: TTY = PT/AB; Suppress PT = "N"; Suppress AB = "Y"
Per above, codes 99 and 999 are considered obsolete, so even though they have an "Inactive" status, they are assigned TTY = OP/OA, Suppress = O (no change from previous version).

Vaccine Status Short Description Suppress Full Vaccine Name TTY Suppress

Exception: Codes 99 and 999 are "Inactive," but are assigned TTY = OP/OA per directions from the CDC

Full Vaccine Name MRCONSO.STR
CODE = CVX Code

SUPPRESS = "N" or "O" depending on value of "Vaccine Status field
Notes MRSAT.ATN = SOS
Vaccine Status MRSAT.ATN = "VACCINE_STATUS"

Vaccine Status is also used to assign TTYs
NonVaccine Not included in Metathesaurus source processing
Last Updated MRSAT.ATN = "DATE_LAST_MODIFIED"

Note:

The NonVaccine field is not included in Metathesaurus source processing. A value of "True" indicates that the concept is a non-vaccines. There are three concepts with NonVaccine = "True," indicating that the concept is not a vaccine:
no vaccine administered
unknown
RESERVED - do not use
All other concepts have a NonVaccine value of "False"
