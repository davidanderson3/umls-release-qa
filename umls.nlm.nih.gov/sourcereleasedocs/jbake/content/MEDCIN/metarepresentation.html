VSAB: MEDCIN3_2023_07_20

Summary of Changes

Mappings to SNOMEDCT_US_2023_06_30

Source Provided Files:

The Metathesaurus uses the following four pipe-delimited MEDCIN files:
  • UMLS_MEDCIN_CONCEPTS2023-07-20.txt
  • UMLS_MEDCIN_DESCRIPTIONS2023-07-20.txt
  • UMLS_MEDCIN_HIERARCHY2023-07-20.txt
  • UMLS_MEDCIN_SNOMED2023-07-20.txt

Identifiers:

Identifiers are assigned as follows:
  • CODE: MEDCIN ID
  • SAUI: Not Applicable
  • SCUI: MEDCIN ID
  • SDUI: Not Applicable

Mappings (MRMAP):

One map set ("XM" atom, along with associated attributes and mappings) is included for MEDCIN.  Mappings are extracted from the file "UMLS_MEDCIN_SNOMEDyy-mm-dd.TXT.  Mappings are created for entries where the RELATIONSHIPTYPE = "168666000" (same_as).
