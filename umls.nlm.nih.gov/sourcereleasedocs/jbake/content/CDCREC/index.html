Authority
The U.S. Centers for Disease Control and Prevention (CDC) curates CDCREC for use in coding race and ethnicity data.

Purpose
The main purpose of the code set is to facilitate use of federal standards for classifying data on race and ethnicity when these data are exchanged, stored, retrieved, or analyzed in electronic form.

Description
CDCREC includes a hierarchy of race terms and a hierarchy of ethnicity terms.

Update Frequency
CDCREC was last updated in 2017.

Metathesaurus Update Frequency
CDCREC was last updated in the Metathesaurus in 2021.
