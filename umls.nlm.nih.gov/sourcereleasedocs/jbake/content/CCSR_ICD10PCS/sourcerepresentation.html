Summary of Changes

CCS_10 has been divided into two sources, CCS_10 (for procedure categories) and CCSR_10 (for diagnosis categories)

Deprecated TTYs:

  • SD (Single-level diagnosis categories) - now included with SAB = CCSR_10
  • MD (Multi-level diagnosis categories) - the file ccs_dx_icd10cm_20##_1.csv was not released so these atoms are not present in CCS_10 or CCSR_10. The new diagnosis mapping file, $DXCCSR2020.CSV, only has mappings to the new single-level CCSR code and is represented with SAB = CCSR_10.

Source Provided Files:

The input files are in standard comma-separated-values format, with fields descriptions in the first line.

File Name Description
ccs_pr_icd10pcs_2030_1.csv Procedure categories and mappings
