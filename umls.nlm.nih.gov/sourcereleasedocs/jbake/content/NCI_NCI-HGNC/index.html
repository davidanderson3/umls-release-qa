Authority
The HUGO Gene Nomenclature Committee (HGNC) maintains the HUGO Gene Nomenclature. The HGNC is a committee of the Human Genome Organisation (HUGO).

Description
The NCI Thesaurus incorporates a small set of symbols from HGNC. These symbols are mapped to thesaurus terms.

For more information about HGNC in the Metathesaurus, see the HGNC source synopsis.

Metathesaurus Update Frequency
The HGNC terms are updated annually as part of the NCI Thesaurus in the Metathesaurus.
