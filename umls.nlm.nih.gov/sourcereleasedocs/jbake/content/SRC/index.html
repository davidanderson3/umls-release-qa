Synopsis Metadata Statistics

Authority
The Unified Medical Language System® (UMLS®) Metathesaurus Source Terminologies is produced by the U.S. National Library of Medicine (NLM).

Purpose
The UMLS Metathesaurus Source Terminologies provides metadata about UMLS Metathesaurus source vocabularies.

Description
The UMLS Metathesaurus Source Terminologies concepts contain information about Metathesaurus source vocabularies, including root and versioned source abbreviations, source restriction levels, and source contact information. Its metadata is found in the MRSAB.RRF file in the Metathesaurus.

Metathesaurus Update Frequency
The UMLS Metathesaurus Source Terminologies is an updated source in each Metathesaurus release.
