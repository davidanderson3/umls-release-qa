Mappings (MRMAP):

MRMAP entries are created by processing data in the MARC 750 (Topical Mappings) tag, where:

  • 2nd indicator value of "2" is a mappng to MSH.
  • Subfield $8 is a "field link/sequence number," indicating that one LCH concept maps to two or more entities in the target source
MRMAP Field
 Origin
MAPSUBSETID
 Value is always "1". Used in conjunction with MAPRANK value to allow for representation when a given FROMEXPR has more than one TOEXPR value in a simple map. This data condition is found in the source where there is more than one "750" tag, and no subfield $8.
MAPRANK
 Value is an integer; incremented when an LCH record has multiple "750" tags, without $8 subfields. This data condition indicates a simple one to many mapping from LCH to MSH (i.e. there is no implied "AND"). The MAPRANK field is used in conjunction with MAPSUBSETID to allow for representation when a given FROMEXPR has more than one TOEXPR value in a simple map.
FROMID

TOEXPR
 The expression of the MSH target entity being mapped to, derived from "750" tag values where subfield $8 is populated, indicating a "field link/sequence number." Includes qualifier information extracted from the $x subfield, if present Note: the original data in this field is populated by MSH strings. Where possible, the strings have been translated to MSH identfiers. In a small number of cases, the string provided did not correspond to a current MSH string, and these entries were dropped during Metathesaurus source processing.
TOTYPE
