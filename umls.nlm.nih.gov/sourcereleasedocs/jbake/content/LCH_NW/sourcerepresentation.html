Source-Provided Files: Details



The data is provided as a MARC 21 format authority file.  Before Metathesaurus source processing, this file was converted to ascii using

file:  lcsh.mrc (converted to lcsh.mrc.txt):

Only the 010, 1XX (Heading Information) and 750 fields (Heading Linking Entry) were processed for inclusion in the Metathesaaurus. Other data such as 4XX (See From Tracing) and 5XX (See Also From Tracing) may be analyzed for inclusion in future versions.

Note:  the original data in this field is populated by MSH strings.  Where possible, the strings have been translated to MSH identfiers.  In a small number of cases, the string provided did not correspond to a current MSH string, and these entries were dropped during Metathesaurus source processing.
