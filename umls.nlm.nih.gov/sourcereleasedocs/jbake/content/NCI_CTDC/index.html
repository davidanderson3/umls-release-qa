Description
Clinical Trial Data Commons (CTDC) provides access to clinical trial data between tumor molecular characterization, treatment, response and progression. The CTDC aggregates data from multiple precision medicine studies that tailor treatment to the specific tumor in an individual patient.

Metathesaurus Update Frequency
CTDC is an updated source in each Metathesaurus release as part of the NCI Thesaurus.
