VSAB: ICNP_2022

Summary of Changes

The submitted file format has changed slightly, and the "ICNP preferred term" string is not necessarily the same as the SNOMEDCT_US PT string. All ICNP PTs are in the same CUI with the SNOMEDCT_US PT with the same code.

Source Provided Files:

The files "ICNP-SCT DC.csv" and "ICNP-SCT IC.csv" consist of a subset of SNOMED CT codes and ICNP preferred terms
