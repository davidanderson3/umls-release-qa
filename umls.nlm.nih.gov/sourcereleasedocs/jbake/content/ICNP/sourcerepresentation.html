Summary of Changes:

There are no changes to the primary ICNP terminological data file format or to UMLS source processing.

Source-Provided Files: Summary

The file "ICNP 2019 Release.xlsx" includes the following tabs:

DC Asserted
IC Asserted
DC Inferred
IC Inferred

The mappings to SNOMEDCT are provided in separate files: "ICNP-SCT Mapping-ICs 2019.csv" and "ICNP-SCT Mapping-DCs 2019.csv"
