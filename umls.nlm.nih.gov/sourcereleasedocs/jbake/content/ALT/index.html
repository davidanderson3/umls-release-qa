Authority
Alternative Billing Concepts (ABC) codes are produced and maintained by ABC Coding Solutions (formerly Alternative Link).

Purpose
ABC codes fill in the gaps of traditional medical code sets and the gaps caused by the retiring of state-specific Healthcare Common Procedure Coding System (HCPCS) III codes. The codes reflect items that are not represented in other code sets.

Description
ABC codes include over 4,500 descriptions of integrative healthcare services, remedies, and supplies. An ABC code is a five letter string followed by a two character code modifier. The string identifies the service, remedy, or supply item. The code modifier identifies the practitioner type.

Audience
ABC codes may be used by health care practitioners and insurers for updating medical records, billing, processing claims, and managing benefits.

Update Frequency
ABC is updated annually.

Metathesaurus Update Frequency
ABC is updated annually in the Metathesaurus, as resources permit.
