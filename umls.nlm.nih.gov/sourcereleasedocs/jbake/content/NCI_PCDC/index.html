Description
Pediatric Cancer Data Commons (PCDC) is a common core data dictionary and common governance structure spanning seven pediatric cancers: neuroblastoma, soft tissue sarcoma, acute myeloid leukemia, acute lymphoblastic leukemia, germ cell tumors, bone tumors, and Hodgkin lymphoma. This dictionary enables cross-disease research and sets a standard for future cancer data commons endeavors.

PCDC is an updated source in each Metathesaurus release as part of the NCI Thesaurus.
