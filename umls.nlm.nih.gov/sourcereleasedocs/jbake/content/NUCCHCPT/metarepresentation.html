Summary of Changes

  • The SAB was changed from NUCCPT to NUCCHCPT.
  • TTY = MP was deprecated because the characteristics file was not processed.

Source Provided Files:

The input files are in standard comma-separated-values (csv) format, with fields descriptions in the first line.

File Name Description
nucc_taxonomy_201.csv Provider taxonomy data file

Atoms (MRCONSO):

Term Type Origin of CODE Origin of STR
PT CODE, SCUI: Code value from nucc_taxonomy_201.csv Created by concatenating <Grouping>; <Classification>, <Specialization> fields
For example, the STR for 101YS0200X is "Behavioral Health & Social Service Providers; Counselor, School"
OP CODE, SCUI: Code value from nucc_taxonomy_201.csv Created by concatenating <Grouping>; <Classification>, <Specialization> fields
For example, the STR for 103GC0700X is "Behavioral Health & Social Service Providers; Clinical Neuropsychologist, Clinical"
TTY = OP is assigned when the "Notes" field indicates that the code is inactive.
