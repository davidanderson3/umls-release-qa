Notes

Summary of Changes:

  (return to top)

There have been significant changes made to the structure of the Perioperative Nursing Data Set (PNDS) and the way codes are assigned since the previous version was processed for inclusion in the Metathesaurus.  PNDS 2nd edition (PNDS2002) included outcomes, diagnoses, interventions, and domains, which were assigned term types of PS/PX, DX, IV, and HT, respectively.  In the current verison, there are assessments, evaluations, implementations, outcomes and domains, and term type assignments have changed accordingly.  Domains are still assigned TTY = "HT," as are placeholder atoms for "Assessment," "Evaluation," "Implementation," and 'Outcome," which were created for the purpose of representing a navigational hierarchy.  All other PNDS atoms are assigned TTY = "PT." 

The domains are the same and have the same codes as the previous version, but the code assignment for everything else has changed, and the strings have often changed also.

Differences between PNDS2002 and PNDS2011 in RRF format are:

1)  The SRL has been changed from 0 to 3 for PNDS.

2)  New term type (TTY) | Suppressible value:
      PT|N

3) Dropped term type (TTY):
      DX
      IV
      PS
      PX

3) Dropped relationship | relationship attribute(RELA):
      RQ|has_outcome

This data is not available in machine-processable form

4) New attribute name (ATN):
      INTERPRETIVE_STATEMENT
      RETIRED_CODE
      SOS

5) Dropped attribute name (ATN):
      INTERVENTION_DESCRIPTION
      INTERVENTION_SEQUENCE
      INTERVENTION_TEXT
      OUTCOME_EXAMPLES
      OUTCOME_INDICATORS
      OUTCOME_STATEMENT

The PNDS distribution includes the following:

Documentation and Reference

The electronic distribution of PNDS did not include documentation or reference material.  However, the printed book, "Perioperative Nusing Data Set 3rd Edition" may provide additional insight into the content and usage of PNDS and information is available here:  http://www.aorn.org/PracticeResources/PNDSAndStandardizedPerioperativeRecord/

Data Files

File Name Description
Revised PNDS codes all_Def_Final_10_29_10  (5).xlsx
 An excel file with 5 tabs:  Assessment, Evaluation, Implementation, Outcome and Domain.  Includes obsolete codes and labels, new codes, new labels, domains, and notes.  Includes definitions and interpretive statements.  This table contains essentially the same content as Chapter 3 in the printed book, however there are minor differences due to editing.


Not included: 

All data in the excel file provided is included in the Metathesaurus except for obsolete labels and notes about code and definition changes.

A PNDS concept record may span multiple rows in the excel format.  A row may represent concept information (code and atom name) or an attribute (definition or interpretive statement).  The RRF representation of data from these tabs is based on the value of column 1, "Obsolete code."
