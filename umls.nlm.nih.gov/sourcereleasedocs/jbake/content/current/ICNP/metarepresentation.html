VSAB: ICNP_2015

Summary of Changes

There are no changes to the primary ICNP terminological data file format or to UMLS source processing.

There are minor changes to the ICNP-SCT mappings:
  • The mapping no longer includes "Mapping Notes."  Therefore, everything was processed as an exact match, there are no longer any RB, RN or XR rels.
  • We now have mappings for both diagnoses (DC) and interventions (IC).  Previously, there were only mappings for DC.
  • The MAPSETXRTARGETID attribute has been deprecated since there are no "No match" values indicated.
