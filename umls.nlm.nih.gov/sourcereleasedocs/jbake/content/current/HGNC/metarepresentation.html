VSAB: HGNC2016_04

Summary of Changes

1. Data fields have been renamed and field order has changed. (See HGNC Source Representational page for details).

2. New data fields and corresponding new ATNs:

  • field: ena; ATN = ENA
  • field: location; ATN = CYTOGENETIC_LOCATION
  • field: location_sortable (not processed)
  • field: gene_family_id; ATN = GENE_FAM_ID
  • field: rgd_id; ATN = RGD_ID

3. Deprecated fields:

  • Chromosome
  • Accession Numbers
  • Specialist Database Links
  • Specialist Database IDs
  • Mouse Genome Database ID (mapped)
  • Entrez Gene ID (mapped)
  • RefSeq (mapped)
  • Ensembl Gene ID (mapped)
  • Vega gene ID (mapped)

4. Source data file now comes in UTF-8 rather than cp1252.
