VSAB: CCSR_10_2020

Summary of Changes

CCSR_10 is a new source in META2020AA. CCS_10 was split into two sources, CCS_10 (for procedure categories) and CCSR_10 (for diagnosis categories)

New TTYs:

  • SD (Single-level diagnosis categories); previously included in CCS_10
  • XM: Cross-mapping set for ICD10CM_2020 to CCSR_10_2020 Mappings

Note: MD (Multi-level diagnosis categories), which were present in CCS_10 are not included in CCSR_10. The file ccs_dx_icd10cm_20##_1.csv was not released so these atoms are not present in CCS_10 or CCSR_10. The new diagnosis mapping file, $DXCCSR2020.CSV, only has mappings to the new single-level CCSR code and is represented with SAB = CCSR_10, TTY = SD.
