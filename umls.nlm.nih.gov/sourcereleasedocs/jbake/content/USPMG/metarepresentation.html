VSAB: USPMG_2020

Summary of Changes

The "Salt/Ester" column has been removed from the original data file. Many of the MRCONSO.STR values that were previously created by concatenating the "Example Drugs" with the "Salt/Ester" column are now included in the "Example Drugs" column (previously labeled "Example Part D Eligible Drugs").
