Description
The NCI Thesaurus incorporates a small set of terms from the Health Level (HL7) Vocabulary.

To learn more about the full HL7 vocabulary as included in the Metathesaurus, see the HL7 source synopsis.

Metathesaurus Update Frequency
NCI-HL7 is updated annually as part of the NCI Thesaurus in the Metathesaurus.
