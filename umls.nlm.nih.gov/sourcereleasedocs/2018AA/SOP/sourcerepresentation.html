Source-Provided Files: Summary

The SOP distribution includes the following. Additional information can be accessed at http://www.phdsc.org/standards/payer-typology.asp

Data files:

File Name Description
SourceofPaymentTypologyVersion7FINALJune27_2016.pdf List of terms in the Source of Payment Typology (converted to .txt file for Metathesaurus source processing)
SourceofPaymentTypologyUsersGuideVersion7FinalJune27_2016.pdf Source of Payment Typology User's Guide

To process SOP, definitions in SourceofPaymentTypologyUsersGuideVersion7FinalJune27_2016.pdf were manually linked to terms in SourceofPaymentTypologyVersion7FINALJune27_2016.pdf

Not included:

All terms in the file SourceofPaymentTypologyVersion7FINALJune27_2016.pdf are processed for the Metathesaurus. Of the definitions in SourceofPaymentTypologyUsersGuideVersion7FinalJune27_2016.pdf, only those for terms in SourceofPaymentTypologyVersion7FINALJune27_2016.pdf are processed for the Metathesaurus.
