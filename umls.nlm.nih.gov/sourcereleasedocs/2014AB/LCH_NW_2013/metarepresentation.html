VSAB: LC_NW_2013

Summary of Changes

LC_NW is a new source.

Source Provided Files:

All files are available here:  http://files.library.northwestern.edu/public/lcshmesh/

Documentation:
    lcsh-mesh_read.doc - Description of the project and files
    lchs-mesh_sample.txt - Sample data
   
    Additional documentation on the MARC 21 Format For Authority Data can be found here:  http://www.loc.gov/marc/authority/

Data: 
   
    lcsh.mrc - MARC 21 format authority file, with linking data to MeSH strings.

Identifiers:

Identifiers are assigned as follows:
  • CODE: Value of the 010 tag (Library of Congress Control Number)
  • SAUI: Not Applicable
  • SCUI: Same as CODE
  • SDUI: Not Applicable
