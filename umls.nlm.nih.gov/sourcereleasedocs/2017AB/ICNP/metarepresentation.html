VSAB: ICNP_2017

Summary of Changes

There are no changes to the primary ICNP terminological data file format or to UMLS source processing.

Source Provided Files:

The file "ICNP 2017 Release.xlsx" includes the following tabs:

DC Asserted
IC Asserted
DC Inferred
IC Inferred

The mappings to SNOMEDCT are provided in a separate spreadsheet:  "ICNP-SCT Mapping 2017.xlsx"

Identifiers:

Identifiers are assigned as follows:
  • CODE: concept code
  • SAUI: N/A
  • SCUI: concept code
  • SDUI: N/A
