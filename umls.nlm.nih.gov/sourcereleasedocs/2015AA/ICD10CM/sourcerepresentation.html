Notes

The following files are downloaded from http://www.cms.gov/Medicare/Coding/ICD10/2015-ICD-10-CM-and-GEMs.html:
  • 2015_I10gem.txt
  • 2015_I9gem.txt
  • reimb_map_dx_2015.txt
  • icd10cm_order_2015.txt
In addition, the Metathesaurus uses files submitted in TREF (Terminology Representation and Exchange Format). These files are produced by the National Center for Health Statistics (NCHS) for Metathesaurus processing, and are not currently available publicly. TREF was developed by the NLM and is a subset of the Rich Release Format (RRF) - the format in which the UMLS Metathesaurus is released. The following files are submitted for ICD10CM:

MRCONSO.TREF
MRDOC.TREF
MRREL.TREF
MRSAB.TREF
MRSAT.TREF

The TREF files align with the publically-available .pdf release of ICD10CM. When relevant, the origin of data from the NCHS .pdf file is noted in the tables below.

Summary of Changes:

(return to top)

There have been no changes to the data format. There were no updates to the atoms (order) file. The mapping data was updated, with different data and slight changes to the format.

Changes for this version are:
  • Change totype from SDUI to BOOLEAN_EXPRESSION_SDUI when there is an AND
  • Fixed bug, changed XRTARGETID from NODX to NOI9
  • Fixed bug in placement of decimal for ICD9CM "E" codes.
