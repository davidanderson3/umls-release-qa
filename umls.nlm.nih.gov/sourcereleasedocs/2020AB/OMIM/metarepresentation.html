VSAB: OMIM2020_07_27

Summary of Changes

No changes were made to the OMIM source format, however significant changes were made to UMLS source processing:

New ATNs:
  • QUALIFIER: Frequency information previously enclosed in parentheses for a PTCS, PHENO or PHENO_ET atom. STYPE = RUI
  • SOS: Previously represented as PTCS atoms treed under "MISCELLANEOUS" or "MOLECULAR BASIS".
Explanation:

QUALIFIER: For atoms with TTY of PTCS, PHENO or PHENO_ET, frequency information was often included parenthetically. Based on pattern matching, this information was stripped from the atom string and is now represented as an attribute on the relationship between the atom and its preferred form (ATN = QUALIFIER). Examples of "stripped" information include "in some patients," "rare," "1 family," and "less common."

Example:
Previous representaton:
  • ATOM: STR = "Doughy or puffy skin (rare)"; TTY = PTCS
  • RELATIONSHIP: RELA = "manifestation_of" to OMIM PT atom "DESBUQUOIS DYSPLASIA 2" (CODE = 615777)
  • ATTRIBUTE: N/A
Current representation:
  • ATOM: STR = "Doughy or puffy skin"; TTY = PTCS
  • RELATIONSHIP: RELA = "manifestation_of" to OMIM PT atom "DESBUQUOIS DYSPLASIA 2" (CODE = 615777)
  • ATTRIBUTE: ATN = "QUALIFIER"; ATV = "rare"; STYPE = "RUI"

SOS: Atoms in the "MISCELLANEOUS" and "MOLECULAR BASIS" branches of the PTCS tree were removed and are now represented as attributes (ATN = SOS; STYPE = AUI)

Examples:
  • Caused by mutations in merlin (NF2, 101000.0001) [MOLECULAR BASIS]
  • Incidence of 1 in 25,000 livebirths [MISCELLANEOUS]

Identifiers:

Identifiers are assigned as follows:
  • CODE:
    • TTY = ACR, ET, ETAL, PHENO, PHENO_ET, PT: <Mim-entry_mimNumber>
    • TTY = PTAV: <Mim-entry_mimNumber>.<Mim-allelic-variant_number> (e.g. "606158.0015")
    • TTY = HT, PTCS: an MTHU# assigned during Metathesaurus source processing
  • SAUI: not applicable
  • SCUI: not applicable
  • SDUI: same as CODE
