VSAB: MSH2011_2010_08_30

Summary of Changes:

No changes have been made to the original data file format.  The following changes have been made in Metathesaurus source processing:

  • Name changes in top levels of hierarchy and MeSH Catogories:

    • "Biological Sciences" is now "Phenomena and Processes"

    • "Natural Sciences is now "Disciplines and Occupations"

    • "Index Medicus Descriptor" is now "Topical Descriptor"

  • Added representation of "<QualifierReferredTo>" tag data for <HeadingMappedTo> tags:
    • For <HeadingMappedTo> tags that include a <QualifierReferredTo> tag, the relationship between the Supplemental Concept and the qualifier was dropped in previous versions.  Now, a "RO|mapping_qualifier_of/has_mapping_qualifier" relationship is created.  To pair the qualifier relationship with the correct mapped_to relationship, the same value for MRREL.RG (relationship group) is assigned.

      •
  • Change in STYPE for AQ/QB relationships:  Relationships between Main Headings and their Allowable Qualifiers are now created with STYPE = "SDUI" instead of STYPE = "SAUI," to better represent MeSH source semantics.

    •
  • Dropped "QA" attribute:  MRSAT.ATN = "QA" is no longer created.  This data is already represented in MRCONSO.RRF as an atom with TTY = "QAB."

Hierarchy (MRHIER):

  (return to top)

The hierarchy is created from values in the <TreeNumber> tags in desc*.xml and qual*.xml.  Official MeSH xml release files do not include the top levels of a hierarchy, so additional top level concepts are created and maintained in separate files.
