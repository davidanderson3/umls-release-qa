VSAB: MEDCIN3_2011_12_15

Summary of Changes

No changes have been made to the original data file format or to the data representation.

Source Provided Files:

The Metathesaurus uses the following four pipe-delimited MEDCIN files:
  • UMLS_MEDCIN_CONCEPTS2011-12-15.TXT
  • UMLS_MEDCIN_DESCRIPTIONS2011-12-15.TXT
  • UMLS_MEDCIN_HIERARCHY2011-12-15.TXT
  • UMLS_MEDCIN_SNOMED2011-12-15.TXT

Identifiers:

Identifiers are assigned as follows:
  • CODE: MEDCINID
  • SAUI: Not Applicable
  • SCUI: MEDCINID
  • SDUI: Not Applicable
