UMLS Source Vocabulary Documentation

Use the tabs below to browse source vocabularies in the current UMLS Metathesaurus.
We update source vocabulary reports following each UMLS release.
Source vocabularies, from UMLS Release 2008AB onward, link to the following reports:

  • Synopsis: General information about the selected source vocabulary. Older source vocabularies contain only a Synopsis.
  • MRSAB.RRF: Source vocabulary contact information, Source Restriction Category, and other Metathesaurus data fields.
  • Statistics and Sample Data: Terms and term types, attributes, relationships, semantic types, and overlap with other source vocabularies.
  • Representation: Details of how the source vocabulary provides its data to the UMLS, and how the UMLS represents the source vocabulary in the Metathesaurus.
