UMLS Quick Start Guide

What is the UMLS?

UMLS is a terminology mapper, dictionary, medical translation tool, coding tool, lexical research tool, medical knowledge source, classification tool...

The UMLS, or Unified Medical Language System, is a set of multi-purpose tools. It brings together many health and biomedical terminologies and organizes synonymous terms into packages, called concepts. This concept structure enables computers to interpret when different terms have the same meaning.

UMLS in Use

One of the powerful and meaningful uses of the UMLS is linking health information, medical terms, drug names, and billing codes across different computer systems. An example is linking common terminology and codes between your doctor, your pharmacy, and your insurance company. Another example is patient care coordination of among several departments within a hospital. The UMLS has many other uses, including search engine retrieval, public health statistics, and terminology research.

Three Parts of the UMLS

The UMLS has three parts, which we call the Knowledge Sources:

  1. The Metathesaurus: Biomedical terms and codes from over 150 vocabularies, including SNOMED CT, LOINC, CPT, MeSH
  2. The Semantic Network: Broad categories (semantic types) and their relationships (semantic relations). Each Metathesaurus concept is assigned to at least one semantic type.
  3. The SPECIALIST Lexicon and Lexical Tools: Natural language processing tools. The SPECIALIST Lexicon includes commonly occurring English words and biomedical terms and many lexical variations for each term. The Lexical Tools address variability in words, such as 'treat,' 'treats' and 'treated.'

How it works?

Over 150 source terminologies included in the UMLS contribute terms and codes, which we call atoms.

How do I do it?

Web browser

Through the UMLS Terminology Services (UTS), you can search the UMLS data through three applications:

  • Metathesaurus browser - Retrieve UMLS concept information, including CUIs, semantic types, and synonymous terms.
  • Semantic Network browser - View the names, definitions, and hierarchical structure of the semantic network.
  • SNOMED CT browser - Search SNOMED CT concepts and descriptions represented in the UMLS.

Local installation

To install the UMLS locally, you can either download the files or request a UMLS DVD through the UTS. Once you have the files on your computer, the MetamorphoSys tool allows you to customize the data according to your needs. You can then load the customized data into your own database system, such as MySQL or Oracle, or browse the data using the MetamorphoSys RRF Browser application.

API queries

You can use our Java-based application programmer interfaces (APIs) to query the UMLS data within your own application.

Get started

To start using the UMLS:

Request a license and sign up for the UMLS Terminology Services (UTS). This account allows you to browse, download, and query all of the UMLS data. The UMLS data is available for free. Some uses of the data require additional agreements with individual terminology vendors.
